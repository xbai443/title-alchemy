# 标题炼金术 · 自媒体爆款标题生成器

> 纯前端 · 无需联网 · 双击即用

## 功能
- 输入主题，一键生成 10-30 个爆款标题
- 6种标题风格：痛点型/数字型/对比型/悬念型/情感型/热点型
- 5个目标平台适配：通用/小红书/公众号/知乎/头条
- 单条复制 / 全部复制 / 导出 CSV

## 使用
双击 `index.html` 即可在浏览器中打开使用。

## 部署到 GitHub Pages
1. 创建 GitHub 仓库
2. 将 `index.html` 上传
3. 开启 GitHub Pages（选择 main 分支）
